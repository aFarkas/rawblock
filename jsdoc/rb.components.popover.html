Class: popover

popover

Creates a popover that is positioned/anchored to another element.

A11y-Notes: If the popover has structured content use the class js-rb-autofocus inside of/at the popover. If it contains simple text use a aria-live="polite" or an appropriate role.

The collision handling. Possible values: 'flip', 'none'. Can be declared separatly for x and y. (i.e. 'flip none')

The anchor element to position the panel against. 'button' means the associated panelbutton module or if no associated panelbutton is found the opening button module. Accepts als string that are processed with rb.elementFromStr.
