Adds a new custom validation rule to the set.

Parameters:
Name Type Description
rule Object
Properties
Name Type Argument Default Description
name String

Name of the custom validation rule.
rule customValidationRuleCallback

The function that runs the validation. Returns false if the rule is not satisfied otherwise false. In case of async validation it has to return a promise/deferred.
message String <optional>
 'This field is invalid.'

Default validation message.
isAsync Boolean <optional>
 false

In case the rule works async.
isDependent Boolean <optional>
 false

In case the rule not only depends on the value of one field.

    name: 'foo',

    }
});
